# TykoonNet

## Overview

TykoonNet is a portfolio project developed as part of the Tykoon Labs engineering portfolio. The objective is to design, build, secure, and document a scalable enterprise network for Tykoon Kreations using Cisco Packet Tracer and industry networking best practices.

---

## Project Goals

- Design a realistic enterprise network
- Configure routing and switching
- Implement VLAN segmentation
- Configure DHCP and DNS
- Secure the network with ACLs
- Document the entire infrastructure
- Practice enterprise troubleshooting
- Build a professional networking portfolio

---

## Technologies

- Cisco Packet Tracer
- Networking Fundamentals
- IPv4 Addressing
- VLANs
- ACLs
- DHCP
- DNS
- NAT
- Wireless Networking
